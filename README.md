## Actividad 1: Aplicación de conversión de soles a dolares
En esta actividad se creó una aplicación que permitiera convertir soles a dólares, en un cuadro para ingresar una cantidad y abajo dos botones, que permitian escoger a cual moneda querías convertir esa cantidad y finalmente un botón de acción para ejecutar la acción
